# Analysis Skill

## Purpose
Ensure decisions are based on evidence, not assumptions.

This skill exists to stop guesswork-driven changes.

## When to Use
Apply this skill when:
- Evaluating SEO changes
- Deciding whether to create new pages
- Reviewing audit results
- Interpreting performance data

## Core Principles
- Evidence before action
- Prefer observation over speculation
- Document conclusions

## Allowed Actions
- Analyse Search Console data
- Identify real performance gaps
- Validate whether changes are needed

## Disallowed Actions
- Acting on “best practice” alone
- Making changes without data or reasoning
- Overreacting to short-term metrics

## Success Indicators
- Fewer unnecessary changes
- Clear rationale for actions taken
- Decisions can be defended later
