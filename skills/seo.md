# SEO Skill

## Purpose
Guide all search engine optimisation work on the site with a focus on
measurable improvements, intent clarity, and crawl efficiency.

This skill exists to prevent repetitive or cosmetic SEO changes.

## When to Use
Apply this skill when working on:
- Page titles, meta descriptions, headings
- Internal linking
- Structured data (schema)
- Crawl/indexation concerns
- Keyword intent ownership

## Core Principles
- Each indexable page must have a single, clear search intent
- Avoid keyword cannibalisation
- Prefer intent separation over copy rewriting
- Optimise for users first, search engines second

## Allowed Actions
- Assign or refine keyword intent per page
- Resolve overlapping or ambiguous intent
- Add structured data that unlocks real SERP features
- Improve internal linking where it clarifies topical relationships

## Disallowed Actions
- Rewriting copy solely “for SEO”
- Making changes without identifying a real problem
- Chasing SEO tool scores
- Repeatedly re-optimising already-correct pages

## Success Indicators
- Clear intent ownership per URL
- Reduced keyword overlap
- Improved crawl clarity
- Increased impressions in Search Console
